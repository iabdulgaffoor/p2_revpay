# RevPay - Modern Payment Platform

## Introduction
RevPay is a secure and efficient digital payment application designed for both individual users and businesses. It enables seamless money transfers, payment requests, and financial management through a user-friendly interface.

## Core Features
- **User Personas**: Supports Personal accounts for everyday transfers and Business accounts for professional transactions.
- **Wallet System**: Each user has a secure digital wallet with real-time balance tracking.
- **Money Transfers**: Quick and secure peer-to-peer (P2P) transfers.
- **Payment Requests**: Users can request money from others, and requestees can approve or decline these requests.
- **Business Features**: Business verification, invoicing, and professional transaction history.
- **Security**: Features multi-factor authentication (MFA), secure transaction PINs, and role-based access control.

## Technology Stack
- **Backend**: Java 17+, Spring Boot, Spring Security, Hibernate/JPA.
- **Database**: H2 (Development) / PostgreSQL (Production ready).
- **Frontend**: Thymeleaf, HTML5, CSS3, JavaScript.

## Getting Started
To run the application locally:
1. Clone the repository.
2. Ensure you have Maven installed.
3. Run `./mvnw spring-boot:run` (or `mvnw.cmd` on Windows).
4. Access the application at `http://localhost:8080`.
